# Contributing

We welcome contributions! Here's how to get started.

## Development Setup

```bash
git clone https://github.com/Deepleaper/opc-deepbrain.git
cd opc-deepbrain
pip install -e ".[dev]"
```

## Running Tests

```bash
pytest tests/
```

## Pull Requests

1. Fork the repo and create your branch from `main`/`master`
2. Add tests for any new functionality
3. Ensure all tests pass
4. Submit your PR with a clear description

## Code Style

- Python: Follow PEP 8
- Use type hints where possible
- Write docstrings for public APIs

## Reporting Issues

- Use GitHub Issues
- Include reproduction steps, Python version, and OS
